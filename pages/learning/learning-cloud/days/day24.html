JSON & REST APIs

Bridging the gap between Go data structures and the web. Master the encoding and decoding of JSON to build modern APIs.

1 JSON Mapping & Struct Tags

Go uses "struct tags" to control how field names appear in JSON output. Fields must be exported (Capitalized) for the JSON package to see them.

2 Encoding & Decoding

Marshal (Struct → JSON)

Converting a Go object into a JSON byte slice.

jsonData, _ := json.Marshal(student)

Unmarshal (JSON → Struct)

Parsing JSON bytes back into a Go struct.

json.Unmarshal(data, &student)

3 The API Handler Pattern

When building APIs, we use Encoders and Decoders to stream JSON directly from the request and response bodies.

// Sending JSON
w.Header().Set("Content-Type", "application/json")
json.NewEncoder(w).Encode(student)

// Receiving JSON
json.NewDecoder(r.Body).Decode(&s)
