Foundations of Neural Networks

If this foundation is strong, deep learning becomes logical. If it is weak, everything later will feel like magic.

1 Biological vs. Artificial

The artificial neuron is inspired by the biological version. It receives signals, processes them, and fires if the total signal strength exceeds a threshold.

Biological

Signals enter through Dendrites, are summed in the Cell Body, and travel down the Axon if threshold is met.

Artificial

Inputs are multiplied by Weights, added to a Bias, and passed through an Activation Function.

2 The Core Equation

Everything in Deep Learning stems from this single formula:

y = f(w · x + b)

x

Inputs (Features)

Raw data points, like pixel values or student GPA.

w

Weights (Importance)

The learned parameters that determine how much each input matters.

b

Bias (Shift)

Allows the model to shift the activation function for better flexibility.

3 Activation: Why Linearity Fails

Real-world data is rarely a straight line. Activation functions add Non-linearity, allowing the network to solve complex problems like image recognition.

// Python implementation (The Pure Way)
def neuron(x, w, b):
    # Weighted Sum (Dot Product)
    z = sum(xi * wi for xi, wi in zip(x, w)) + b
    
    # Simple threshold activation
    return 1 if z > 0 else 0

# Example
inputs = [2, 3]
weights = [0.5, 1.0]
bias = 1
print(neuron(inputs, weights, bias)) # Result: 1
