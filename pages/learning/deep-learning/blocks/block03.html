Artificial Neural Networks

This is where single neurons become intelligence. We move from individual units to a stacked architecture capable of modeling the world.

1 ANN Anatomy

An ANN is a hierarchical stack of layers. Each layer extracts progressively more complex features from the data.

Input Layer

Holds raw data (pixels, features). No computation happens here; it's simply the entry point.

Hidden Layers

Where the "Deep" in Deep Learning lives. These layers learn abstract feature representations.

Output Layer

Produces the final prediction. Activation depends on task (Sigmoid, Softmax, or Linear).

2 Forward Propagation

Forward propagation is the process of data flowing from input to output to make a prediction.

$a^{(l)} = f(W^{(l)} a^{(l-1)} + b^{(l)})$

The output of layer $(l-1)$ is the input to layer $(l)$.

// PyTorch Conceptual Structure
model = nn.Sequential(
    nn.Linear(2, 4),  # Input layer (2) to Hidden (4)
    nn.ReLU(),        # Non-linear activation
    nn.Linear(4, 1),  # Hidden (4) to Output (1)
    nn.Sigmoid()      # Binary classification output
)

3 Depth vs Width

Width

Number of neurons in a layer. More width allows learning more features at the same level of abstraction.

Depth

Number of layers. More depth allows for hierarchical learning (e.g., Layer 1: edges, Layer 2: shapes, Layer 3: faces).

XOR Problem Insights

A single neuron cannot solve XOR because the data is not linearly separable. An ANN solves it by using a hidden layer to "bend" the feature space, allowing the output layer to separate the classes with a simple line.
