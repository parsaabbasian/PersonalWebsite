Block 4 (2-3 Days)

Loss & Optimization

How does the network know it is wrong, and how does it improve? This block separates those who build models from those who understand training.

1 The Learning Loop

A neural network learns by making a prediction, measuring how "wrong" it is via a Loss Function, and then using an Optimizer to adjust weights and reduce that error.

Loss Function

"The Scorecard: Tells us how far we are from the truth."

Optimizer

"The Guide: Tells us how to change weights to improve the score."

2 Essential Loss Functions

Mean Squared Error (MSE)

Common in Regression. Punishes large errors heavily.

$MSE = \frac{1}{n} \sum (y - \hat{y})^2$

Binary Cross-Entropy (Log Loss)

Standard for Binary Classification. Penalizes confident wrong predictions.

$L = -[y\log(\hat{y}) + (1-y)\log(1-\hat{y})]$

3 Gradient Descent & Learning Rate

Optimization is like finding the bottom of a valley in the dark. We move weights in the direction that reduces loss: $w = w - \eta \cdot \nabla L$.

The Learning Rate ($\eta$) Strategy:

  • Too Small: Excruciatingly slow convergence.
  • Too Large: Overshooting the minimum, divergence, and chaos.
  • Adam Optimizer: Adaptive step sizes (The Industry Default).
