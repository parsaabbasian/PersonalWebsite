Day 22

Go Modules (go mod)

Mastering dependency management, versioning, and external packages in professional Go environments.

1 What Is a Go Module?

A Go module is the standard way to manage dependencies. It consists of a collection of Go packages stored in a file tree with a go.mod file at its root.

Initialization

go mod init my-project

2 Managing Dependencies

go.mod

Lists the module path and the specific versions of your dependencies.

module my-project
go 1.22
require github.com/google/uuid v1.6.0

go.sum

Contains cryptographic hashes to ensure dependency integrity across builds.

github.com/google/uuid ... h1:NI...

3 The go mod Toolkit

Command Purpose
go mod tidy Removes unused deps and adds missing ones.
go mod download Downloads dependencies to local cache.
go list -m all Lists the current module and all dependencies.
