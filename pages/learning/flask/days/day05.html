2 Metadata: Headers & Cookies

Headers

The "Instructions." They carry metadata like Content-Type or Authorization tokens.

Cookies

The "Memory." Stored in the browser to maintain state (like sessions) across independent requests.

3 Professional Responses

A response tells the client exactly what happened using a Status Code and provides the Body (HTML/JSON).

Common Status Codes
200 OK (Success)
201 Created (Success)
400 Bad Request (Client Error)
404 Not Found (Client Error)
500 Server Error (Backend Bug)
