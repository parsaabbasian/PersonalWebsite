Day 23

HTTP Servers in Go

Discovering Go's built-in web capabilities. Build production-ready servers using the robust net/http package.

1 The Power of net/http

Go includes a world-class HTTP library in its standard library. Unlike Python or Node.js, you often don't need a framework to build high-performance APIs.

The Basic Loop

func handler(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintln(w, "Hello World")
}

func main() {
    http.HandleFunc("/", handler)
    http.ListenAndServe(":8080", nil)
}

2 Request vs. Response

Request (*http.Request)

Contains all data sent by the client: URL, Methods, Headers, and Query Parameters.

r.URL.Query().Get("id")

Response (http.ResponseWriter)

An interface used to construct the HTTP response: Status Codes and Body content.

w.WriteHeader(404)

3 Routes & HTTP Methods

You can define multiple endpoints and restrict them by HTTP Method (GET, POST, etc.).

if r.Method != http.MethodGet {
    http.Error(w, "Only GET allowed", http.StatusMethodNotAllowed)
    return
}
