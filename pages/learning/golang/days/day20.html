Standard Library Basics

Exploring Go's "batteries-included" philosophy through core packages for time, OS interaction, and type conversion.

1 The time Package ‚è±Ô∏è

Delays & Measurement

time.Sleep(2 * time.Second)

start := time.Now()
elapsed := time.Since(start)

Key Utility

The time package is essential for handling timeouts in concurrency, measuring performance, and formatting dates.

2 The os Package (Environment)

Interact with your Operating System by reading environment variables and command-line arguments.

// Read Env Var
db := os.Getenv("DB_NAME")

// Read CLI Args
args := os.Args // [app_name arg1 arg2]

Note: os.Args[0] is always the name of the program itself.

3 The strconv Package üî¢

Crucial for converting strings to numeric types and vice-versa, especially when handling user input or CLI arguments.

String to Int

strconv.Atoi("42")

Int to String

strconv.Itoa(100)

Parse Float

strconv.ParseFloat("7.5", 64)
