Day 2

Variables & Data Types

Mastering variable declarations, basic types, and the difference between var and :=.

1 Declaring Variables

Method 1: The 'var' keyword

Explicit and clear. Used when you want type clarity or are declaring variables outside functions.

var age int = 20
var name string = "Parsa"

Method 2: Short Declaration (:=)

The most common way. Go automatically detects the type.

city := "Toronto" // Go detects string
gpa := 7.75 // Go detects float64

Note: := only works inside functions and cannot be used for reassignment.

2 Basic Data Types

Type Example Description
int 10 Whole numbers (Integers)
float64 7.75 Decimal numbers
string "Hello" Textual data
bool true Logical true or false

3 Constants & Clean Code

Constants (const)

Values that never change.

const pi = 3.14
// pi = 3.15 -> Error!

Multiple Declaration

Keep your code idiomatic and clean.

x, y := 10, 20
name, age := "Go", 14
