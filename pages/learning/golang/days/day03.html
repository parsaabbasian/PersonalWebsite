Control Flow in Go

Mastering program logic using conditional statements, switch cases, and logical operators.

1 The if / else Statement

Basic Conditional

Go conditions do not use parentheses, and curly braces are strictly mandatory.

age := 20
if age >= 18 {
    fmt.Println("You are an adult.")
}

if with Initialization (Go-Style ‚≠ê)

Declare a variable directly in the if statement. It only exists within that scope.

if x := 10; x > 5 {
    fmt.Println("x is greater than 5")
}

3 The switch Statement

Go is cleaner: no break is needed between cases.

day := 3
switch day {
case 1:
    fmt.Println("Monday")
case 2, 3:
    fmt.Println("Tuesday or Wednesday")
default:
    fmt.Println("Invalid")
}
