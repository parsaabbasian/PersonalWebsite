2 Taking User Input

Basic input using fmt.Scan

var name string
fmt.Print("Enter your name: ")
fmt.Scan(&name)

Pointer Required: The & symbol provides the memory address, allowing Scan to modify the variable directly.

3 Interactive Program Example

calculator.go
switch op {
case "+":
    fmt.Println("Result:", a+b)
case "/":
    if b == 0 {
        fmt.Println("Cannot divide by zero")
    } else {
        fmt.Println("Result:", a/b)
    }
}
