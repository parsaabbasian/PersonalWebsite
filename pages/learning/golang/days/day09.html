Day 9

Maps & Range

Mastering key-value pairs in Go: declaration, manipulation, and the idiomatic way to check for existence.

1 Declaring Maps

Method 1: Using make

The most common way to initialize an empty map.

ages := make(map[string]int)
ages["Parsa"] = 20

Method 2: Map Literal

Declare and initialize with data simultaneously.

grades := map[string]float64{
    "Math": 7.8,
    "CS": 8.4,
}

2 The "Comma OK" Idiom

In Go, accessing a missing key returns a zero value. Always use the "ok" pattern to check if a key actually exists.

value, ok := grades["Physics"]

if ok {
    fmt.Println("Found:", value)
} else {
    fmt.Println("Key not found")
}

3 Operations & Iteration

Update & Delete

grades["Math"] = 8.0 // Update
delete(grades, "CS") // Delete

Looping with Range

for key, value := range grades {
    fmt.Println(key, value)
}

Note: Iteration order in maps is randomized and not guaranteed.
