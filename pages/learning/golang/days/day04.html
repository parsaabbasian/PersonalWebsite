Day 4

Loops in Go

Understanding all forms of the for loop, flow control with break/continue, and how Go simplifies iteration.

1 The "for" Everything Loop

1. Classic for Loop

The standard initialization, condition, and update pattern.

for i := 0; i < 5; i++ {
    fmt.Println(i)
}

2. for as a "while" Loop

Go does not have a while keyword. Simply omit the initialization and update.

i := 0
for i < 5 {
    fmt.Println(i)
    i++
}

3. Infinite Loop

Runs forever until a break or return is encountered.

for {
    fmt.Println("Running forever")
}

2 Break & Continue

break

Exits the loop immediately.

if i == 5 {
    break
}

continue

Skips the current iteration.

if i == 3 {
    continue
}
