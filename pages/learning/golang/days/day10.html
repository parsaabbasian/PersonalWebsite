Day 10

Structs in Go

Building custom data types to group related data together into clean, real-world objects.

1 Defining & Creating Structs

1. Define the Structure

Group related fields under one name.

type Student struct {
    Name string
    Age int
    GPA float64
}

2. Create a Literal

Initialize values using a struct literal.

s := Student{
    Name: "Parsa",
    Age: 20,
    GPA: 7.75,
}

2 Slice of Structs

This is the most common pattern in Go for handling lists of objects, like database results or JSON arrays.

students := []Student{
    {Name: "Parsa", Age: 20, GPA: 7.75},
    {Name: "Sara", Age: 21, GPA: 8.2},
}

for _, s := range students {
    fmt.Println(s.Name, s.GPA)
}

3 Philosophy: Composition over Inheritance

No Classes

Go keeps it simple by avoiding complex hierarchy.

Predictable

Fields are explicit and easy to track.

Composition

Build complex objects by nesting structs.
