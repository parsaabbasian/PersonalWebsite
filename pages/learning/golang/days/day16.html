Day 1

Go Installation & First Program

Setting up your environment and understanding the anatomy of a Go application.

1 Installation & Verification

Step 1: Install Go

Go to go.dev/dl and download the version for your OS. Follow the standard installation wizard (next → finish).

Step 2: Verify Installation

Open your Terminal or PowerShell and run:

go version

Expected output: go version go1.22.x windows/amd64 (or similar)

2 Creating the Project

  1. 1
    Create a folder:
    mkdir go-day1
    cd go-day1
  2. 2
    Create the file:

    Inside the folder, create a file named main.go and paste this code:

    main.go
    package main

    import "fmt"

    func main() {
        fmt.Println("Hello, Parsa! Welcome to Go.")
    }

3 Line-by-Line Breakdown

package main

Every Go file starts with a package declaration. main tells Go this is an executable program, not just a library.

import "fmt"

Short for Format. Itâ€™s a standard library used for printing text to the console and handling input/output.

func main()

The Entry Point. When you run the program, the computer starts executing instructions from inside these curly braces.

4 Run & Build

Method A: Quick Run (Development)

Compiles and runs the code instantly without saving a binary.

go run main.go

Method B: Build (Production)

Creates a standalone .exe or binary file you can give to anyone.

go build
# To run the created file:
./main
