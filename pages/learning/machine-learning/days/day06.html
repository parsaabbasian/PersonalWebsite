2 The DRY Principle: Inheritance

Instead of copy-pasting your navbar and footer, use Template Inheritance to create a base layout that other pages simply "fill in."

# base.html (The Parent)
<nav>...</nav>
{% block content %}{% endblock %}
<footer>...</footer>
# home.html (The Child)
{% extends "base.html" %}
{% block content %}
<h1>Welcome to Home</h1>
{% endblock %}
