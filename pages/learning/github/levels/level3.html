Level 3

The Collaborator

Working with teams, mastering the Pull Request, and resolving the dreaded Merge Conflict.

1 The Team Workflow Concepts

Clone

"Download" an entire project from the cloud to your computer for the first time.

Pull

"Update" your computer with the changes your teammates pushed to the cloud.

PR

"Asking permission." You don't just shove code into the main codebase; you ask the team to review it first.

2 The Simulation (Let's Break Things)

We are going to simulate a situation where you and a "teammate" edit the same file at the same time.

Step 1: The Teammate (Edit on GitHub)

Go to GitHub.com, edit index.html, and commit this change:

<h1>Welcome to the BEST Website</h1>

Step 2: You (Edit Locally)

Do NOT pull yet. Edit index.html on your computer differently:

<h1>Welcome to MY COOL Website</h1>

Then save and commit:

git add .
git commit -m "Updated title locally"

3 The Collision & Resolution

1. The Crash (Push)

When you try to push, Git panics.

git push
! [rejected] error: failed to push some refs...

2. The Fix (Pull)

Download the conflicting changes.

git pull

CONFLICT (content): Merge conflict in index.html

3. The Markers (Open the File)

Open index.html. You will see this mess:

<<<<<<< HEAD
<h1>Welcome to MY COOL Website</h1>
=======
<h1>Welcome to the BEST Website</h1>
>>>>>>> a1b2c3...

4 Forking & Pull Requests

How to contribute to open source (e.g., NASA, Facebook) without working there.

  1. Fork: Click the "Fork" button on their GitHub page. This copies their repo to your profile.
  2. Clone & Edit: Download your fork, make changes, and push back to your fork.
  3. Pull Request: Click "New Pull Request". This sends a message to the original owner: "Hey, I fixed a bug. Want to merge it?"

Mastery Notes: Cheat Sheet

The Golden Rule

ALWAYS git pull before you start working!

2. New Commands

git clone <URL> // Download repo
git pull origin main // Update local

3. Resolving Conflicts Workflow

1. git pull // Get conflict
2. Fix code // Delete markers
3. git add .
