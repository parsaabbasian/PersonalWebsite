C++ Basics: Part 1

1 Your First Program

Every C++ program starts with a "skeleton." Let's look at the most basic code to print text to the screen.

#include <iostream> // 1. Access to input/output library

int main() {        // 2. The entry point of the program
    std::cout << "Hello, World!"; // 3. Print to screen
    return 0;       // 4. End the program successfully
}

2 Variables and Data Types

3 User Input

#include <iostream>

int main() {
    int age;
    std::cout << "Enter your age: ";
    std::cin >> age; // Takes input from keyboard
    std::cout << "You are " << age << " years old!";
    return 0;
}

4 Making Decisions (If/Else)

#include <iostream>

int main() {
    int score = 85;
    if (score >= 90) { std::cout << "Excellent job!"; } 
    else if (score >= 50) { std::cout << "You passed."; } 
    else { std::cout << "You failed. Try again."; }
    return 0;
}

5 Loops

The While Loop

int i = 0;
while (i < 5) {
    std::cout << i << "\n";
    i++; // This increases i by 1
}

The For Loop

// structure: (start; stop condition; step)
for (int i = 0; i < 5; i++) {
    std::cout << "Count is: " << i << "\n";
}
